# Email Block

Send emails using resend.

## Configuration
- **Provider**: resend
- **Format**: html
- **API Route**: `/api/email-send-847a5288`

## Setup

1. Set the required environment variable(s) for your provider.
2. Use the `sendEmail` function or hit the `/api/email-send-847a5288` endpoint.

## Usage

```typescript
import { sendEmail } from '@/lib/email-smtp/lib/email-sender';

const result = await sendEmail({
  to: 'user@example.com',
  subject: 'Hello',
  body: '<h1>Hello World</h1>',
  bodyFormat: 'html',
});
```
